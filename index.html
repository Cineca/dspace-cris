Welcome!

dspace-cris is a new additional open source module for the DSpace platform (http://www.dspace.org). It extends the DSpace data model giving the ability to manage, collect and expose data about all the Research aspects (people, organization units, prize, project, grants, etc.).

The project aims to make publicly available the feature initially developed by CILEA for the institutional repository of The University of Hong Kong, The HKU Scholars Hub

The source code is now available in our public Git Repository! Check the detailed roadmap

As this project is a module for the DSpace platform be sure to check the DSpace CILEA fork, the branch dspace_1_8_x_cris provide out-of-box integration of the DSpace CRIS module
